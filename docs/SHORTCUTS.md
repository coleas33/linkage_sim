# Keyboard and Mouse Shortcuts

## Keyboard Shortcuts

| Shortcut | Context | Action |
|----------|---------|--------|
| `Ctrl+Z` | Global | Undo |
| `Ctrl+Y` or `Ctrl+Shift+Z` | Global | Redo |
| `Ctrl+N` | Global | New empty mechanism |
| `Ctrl+S` | Global (native only) | Quick save to last file path, or Save As if no path yet |
| `Ctrl+Shift+S` | Global (native only) | Save As (always shows file dialog) |
| `Delete` or `Backspace` | Global | Delete the selected body or joint |
| `F` | Canvas | Fit all bodies into the viewport |
| `Escape` | Canvas | Cancel the active tool (Draw Link, Add Body) and return to Select mode |
| `Enter` | Canvas (Add Body tool) | Finish placing the current body (requires 2+ points) |

## Mouse Controls

| Input | Context | Action |
|-------|---------|--------|
| Left-click drag | Canvas (Select mode, empty space) | Pan the view |
| Left-click drag | Canvas (Select mode, near attachment point) | Move the attachment point |
| Middle-click drag | Canvas | Pan the view |
| Right-click drag | Canvas | Pan the view |
| Shift + left-click drag | Canvas | Pan the view |
| Scroll wheel | Canvas | Zoom toward / away from the mouse cursor |
| Left-click | Canvas (Draw Link tool) | Start drawing a new link; snaps to nearby attachment points |
| Left-click release | Canvas (Draw Link tool) | Finish the link at release position; auto-creates joints at endpoints |
| Left-click | Canvas (Add Body tool) | Place a point for the new body |
| Double-click | Canvas (Add Body tool) | Finish placing the current body (same as Enter) |
| Right-click | Canvas | Context menu (suppressed if right-drag was used to pan) |
